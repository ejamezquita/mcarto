# Topological Data Analysis (TDA) to characterize spatial distributions

## Looking at mRNA localization at sub-cellular resolution

**Code Author:** Erik J. Am&eacute;zquita (only responsible for code found in this repository)

**Last substantial modification:** September 2025

---

This repository corresponds to the code used to compute all the TDA-related results described in:

> Tennant S., Amézquita E.J., Smith B., Subramanya S.S.M.V., Cervantes-Pérez S.A., Thibivilliers S., Bhattacharya S., Klaver J., Libault M. (2025) **The Differential Subcellular Localization of Soybean Transcripts, an Additional Regulatory Mechanism of Gene Activity**. [DOI: _to be added_]()

Please refer to the above manuscript for more details on the data motivation, acquisition, formatting, storage, and interpretation.

---

## How to use this repository

All the code and its motivation is described as a series of Jupyter notebooks in the `jupyter` directory. This notebooks were then translated to `.py` scripts to be run for all the data.

---

# Requirements

The code was written in a Linux machine with Python 3.10.
